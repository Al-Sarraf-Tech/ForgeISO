# .GEMINI.md - ForgeISO

This is a complex Rust workspace that builds tools for creating customized, unattended Linux installation ISOs. The project is separated into a core `engine` crate and multiple UI crates for the CLI, TUI, and desktop GUIs.

### Workspace Crates
- `engine`: The core library containing all logic for ISO inspection, modification, and generation.
- `cli`: The primary command-line interface (`forgeiso`).
- `tui`: A terminal UI powered by Ratatui (`forgeiso-tui`).
- `forge-slint`: The primary desktop GUI, built with Slint (`forge-slint`).
- `forge-gui`: An alternative desktop GUI, built with egui (`forge-gui`).

### Core Commands
This host has 18 cores; use them for builds and tests.
- `export CARGO_BUILD_JOBS=18`
- `cargo build --workspace --release`: Build all crates in release mode.
- `cargo run -p forgeiso-cli -- --help`: Run the CLI tool from source.
- `cargo run -p forgeiso-tui`: Run the TUI from source.
- `cargo run -p forge-slint`: Run the primary GUI from source.

### Validation Commands
- **Full Local CI (Critical):** The most important validation step is to run the full CI pipeline locally using Docker. This matches the pre-push git hook.
  ```bash
  make ci-local
  ```
  This command runs a 7-stage pipeline in parallel, covering formatting, linting, unit tests, security audits, and end-to-end ISO build/boot tests.

- **Individual Checks:**
  - `cargo test --workspace`: Run all Rust unit and integration tests.
  - `cargo clippy --workspace --all-targets -- -D warnings`: Run the linter, treating all warnings as errors.
  - `cargo fmt --all --check`: Check for formatting issues.
  - `cargo deny check`: Check for license and security advisory violations.

### Key Architectural Points
- The `ForgeIsoEngine` struct in `engine/src/orchestrator.rs` is the single entry point for all operations.
- All configuration options are defined in structs within `engine/src/config.rs`.
- `AGENTS.md` is the authoritative source of truth for project architecture and conventions. Refer to it for detailed information before making significant changes.
