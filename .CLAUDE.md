# Last updated: 2026-03-11 | Evidence: Cargo.toml, engine/src/*.rs, Makefile, AGENTS.md, scripts/

## What It Is
Rust workspace (v0.2.1) that takes a stock Linux ISO and produces a customized
unattended-install ISO. Supports Ubuntu (cloud-init), Fedora/RHEL (Kickstart),
Arch (archinstall JSON), Linux Mint (Calamares preseed).

## Workspace Members (root Cargo.toml)
| Crate | Binary | Role |
|---|---|---|
| `engine` | (library) | All ISO logic — the only place platform logic lives |
| `cli` | `forgeiso` | Thin Clap CLI → engine calls |
| `tui` | `forgeiso-tui` | Ratatui terminal dashboard |
| `forge-gui` | `forge-gui` | egui/eframe 0.33 desktop GUI (patched egui) |
| `forge-slint` | `forge-slint` | Slint 1.15 desktop GUI (current primary GUI) |

**Note:** `gui/` (legacy Tauri/React) is a separate Cargo workspace. Built by C3 CI stage.
It has its own version in `gui/package.json`, `gui/src-tauri/Cargo.toml`, `gui/src-tauri/tauri.conf.json`.

## Build / Test (18 cores on this host)
```bash
export CARGO_BUILD_JOBS=18

# Development
cargo run -p forgeiso-cli -- doctor   # CLI
cargo run -p forgeiso-tui             # TUI

# CI gate (must be clean before push)
cargo fmt --all --check
cargo clippy --workspace --all-targets -j 18 -- -D warnings
cargo test --workspace -j 18
cargo deny check

# Local CI (all 7 stages in parallel ephemeral containers)
make ci-local       # alias for ci-parallel
make ci-parallel    # scripts/ci/run-parallel.sh
make ci-base        # build C0 warm-cache base image (run once)
```

## Branch Protection
`main` is protected. Never push directly. Feature branch → PR → squash merge:
```bash
gh pr merge <N> --squash --admin --delete-branch
```

## Security Rules (non-negotiable)
- Never serialize passwords to disk. All password fields: `#[serde(skip)]`.
- Always use `workspace.safe_join()` — NEVER raw `PathBuf::join` with user input.
- SHA-256 verification applies to the SOURCE ISO only, never the output ISO.
- `InjectConfig::validate()` must reject unsafe input on all structured fields.
